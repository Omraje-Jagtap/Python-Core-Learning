## 📁 Repository Architecture

My work is organized into logical sub-folders to maintain clean engineering architecture:

* **`/basic-program`**: Foundational logic, syntax, and control flow practice using loops and conditions.
* **`/recursion`**: Exploring recursive function logic, understanding base cases, and breaking down iterative patterns.
* **`/function`**: Modular code blocks tracking functional logic, argument passing, and clean value execution arrays.
* **`/OOP`**: Practical code implementing the pillars of Object-Oriented Programming (Encapsulation, Inheritance, Polymorphism, and Abstraction) to understand how advanced ML frameworks function under the hood.
* **`/file-IO`**: Pipeline scripts parsing external data files (like `.txt` or `.csv` arrays) and implementing crash-safe `try...except` exception handling.
